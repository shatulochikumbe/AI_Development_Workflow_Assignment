# AI_Development_Workflow_Assignment

This repository contains all work for the AI Development Workflow Assignment
for the PLP Software Engineering Program.

## Contents
- Full project PDF (in /docs)
- Example datasets (in /src/data)
- Preprocessing scripts
- Model training scripts
- Evaluation scripts
- Deployment example API
- Jupyter notebooks for exploration

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run preprocessing scripts:
   python src/preprocessing/preprocess_students.py

3. Train models:
   python src/models/train_student_dropout_model.py

4. Run evaluation:
   python src/evaluation/evaluate_students.py

5. Launch API:
   python src/deployment/api.py

## Technologies
- Python
- Scikit-learn
- XGBoost
- FastAPI
- Pandas, NumPy
